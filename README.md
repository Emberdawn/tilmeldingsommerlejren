# tilmeldingsommerlejren
Et WordPress-plugin til sommerlejr-tilmeldinger for registrerede brugere.

## Funktioner
- Shortcode: `[summer_camp_registration]`.
- Gemmer seneste indtastede tilmeldingsdata pr. bruger.
- Gem-knap til kladde og automatisk visning af "Send til godkendelse" når alle krav er opfyldt.
- Prisberegning baseret på voksne/børn og antal dage.
- Upload af screenshot/kvittering for overførsel.
- Admin-menu med prisindstillinger.
- Admin-tabeller for alle tilmeldinger og ikke-godkendte tilmeldinger.
- Ikke-godkendte tilmeldinger kan godkendes i backend.
- Screenshot kan åbnes i en stor lightbox med zoom ved klik.

## Installation
1. Læg `summer-camp-registration.php` i `wp-content/plugins/` (eller i en plugin-mappe).
2. Aktivér pluginet i WordPress.
3. Indsæt shortcode `[summer_camp_registration]` på en side.
4. Sæt priser i backend under **Sommerlejr → Prisindstillinger**.
